# boloniocoazzoli.com

## Running it locally

1.  **Fork and clone** the repo on GitHub.
2.  **Configure and install** the dependencies.

```
$ npm install
```

3.  **Run** the project.

```
$ npm run dev
```

4. You can **open** `http://localhost:8000/` in your browser
